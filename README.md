# FLO Azerbaijan — Enterprise Fashion‑Tech E‑Commerce Platform

Premium, mobile‑first, multi‑language e‑commerce experience built for **FLO Azerbaijan** with an enterprise‑grade back office and a native‑app‑like UX on the storefront.

## Project Vision

FLO Azerbaijan is designed as a high‑performance **Fashion‑Tech** platform that blends editorial, premium UI with a scalable enterprise architecture. The goal is a storefront that feels like a native app on mobile, while keeping operations powerful, auditable, and extensible.

## Tech Stack (TALL)

- Laravel 11
- Filament PHP v3 (Enterprise Admin)
- Livewire v3
- Alpine.js
- Tailwind CSS

## Key Features

- 🌍 Multi‑language: AZ / RU / EN
- 🧠 Omnichannel architecture (storefront + enterprise admin)
- 📱 Mobile‑First “Native App” UX patterns (bottom nav, touch‑first layouts)
- 🧩 Advanced variant management (colors, sizes, stock)
- 🛒 Real‑time cart system (Livewire powered)
